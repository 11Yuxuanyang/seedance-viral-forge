How to make 即梦 merchant dance video scripts

Good merchant dance videos are not about low-grade attraction. They are about using character energy to pull the viewer into a place worth visiting.

Home Merchant dance skill

The actual job

The job is local traffic conversion. A strong script uses the character as the stopping trigger, but the venue remains visible enough that the viewer wants to ask where it is, what it sells, or whether the atmosphere matches a weekend plan.

How this repository handles it

The merchant dance skill asks for the merchant type, the most filmable scene, and the offer. It then proposes three visual directions with different character and rhythm choices before writing the final 15-second script.

Why scene detail matters

If the prompt only says “in a cafe” or “inside a hotpot restaurant,” the output collapses into generic imagery. This repository pushes operators to define space, lighting, signature elements, and movement paths so the generated video actually sells the place.
