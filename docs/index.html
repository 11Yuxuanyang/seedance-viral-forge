Generate Chinese short-video scripts for products that need attention, story logic, reveal timing, and a natural CTA.

Merchant dance lead generation

Reusable prompt operations

Study how production prompt work can be encoded into `SKILL.md` files with inputs, defaults, and hard constraints.

Who should use this

Use this repo if you are an operator, creator, agency, merchant, or builder trying to make AI video output more usable in actual content production. It is especially relevant when the question is not “how do I write a prompt” but “how do I generate a script that survives contact with publishing?”
