# AI带货视频助手

全场景AI视频脚本生成系统。基于 Seedance 2.0（即梦），覆盖带货、引流、流量三大场景，输出可直接复制到即梦平台的提示词。

## Skills

| Skill | 触发 | 场景 | 说明 |
|-------|------|------|------|
| 通用场景 | `/通用场景` 或 "抖音脚本""带货视频""出脚本" | 产品带货 | 给产品信息 → 3个方案 → 15秒分镜脚本。支持批量矩阵（5打法×20角度=100条）|
| 跳舞 | `/跳舞` 或 "跳舞视频""商家跳舞""舞蹈引流" | 商家引流 | 给商家信息 → 小姐姐/帅哥在店里跳舞 → 15秒分镜脚本。支持批量矩阵 |
| 热梗 | `/热梗` | 纯流量 | 自动抓热点 → 匹配爆款模式 → 即梦提示词。不带货，只搞播放量 |

## 核心原则

- **去掉商品链接，观众也愿意看完** — 所有脚本的底线标准
- **即梦直出** — 脚本可直接复制到即梦，不需要二次编辑
- **全中文提示词** — Seedance 2.0 对中文理解更好
- **一镜到底优先** — Seedance 单条生成最连贯
- **脚本≤2000字** — 即梦输入限制

## 输出规范

### 带货/跳舞脚本（通用场景 & 跳舞）

```
产品/场景使用方式说明：[≤200字]

竖屏9:16，15秒，[风格]，[色调]

【0-2秒】
画面：[50-80字]
台词：[角色（情绪）："台词"，无则省略]
声音：[环境音+音效]
字幕：[内容，无则省略]

...（共7-8段）

整体：[一镜到底/场景切换说明]
禁止：任何LOGO或水印

标题：[≤30字，带emoji]
```

### 流量脚本（热梗）

```
参数：竖屏9:16，[5-15秒]，[风格]

[一整段即梦提示词，不分beat]

标题：[≤55字，带emoji]
话题：[10-15个标签]
```

### 文件存储

- 带货脚本：`{产品名}/脚本{N}-{打法}-{角度}.md`
- 跳舞脚本：`{商家名}/脚本{N}-{打法}-{角色类型}.md`
- 流量脚本：`流量视频/{热点关键词}/脚本{N}-{角度}.md`

## 热梗爆款公式

> 平台红利 × 热点/热梗 × 未被满足的想象 × 已验证模式 = 爆款

| 变量 | 含义 |
|------|------|
| 平台红利 | 即梦是抖音的，抖音给流量倾斜 |
| 热点/热梗 | 当下正火的人物、事件、梗 |
| 未被满足的想象 | 大众渴望看到但从没人做过的画面（AI才能做） |
| 已验证模式 | 名人打篮球、热梗视觉化、名人对决、角色穿越、平行宇宙、爆款复刻 |

## Seedance 提示词通用规范

- 每次人物/产品出场都要**重复核心外貌/物理细节**（Seedance不跨段记忆）
- 动作描述要写明**物理机制**（不能只写"用了产品"，要写具体怎么操作）
- 镜头语言写明**景别+运镜+角度**
- **禁止品牌名**出现在提示词中（Seedance无法生成品牌标识），品牌名只放标题
- **禁止价格数字**出现在脚本中（价格由商品链接展示）
- **禁止LOGO和水印**
- 声音描述包含环境音+音效+氛围音乐
